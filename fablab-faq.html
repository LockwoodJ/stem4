Common Questions

These answers apply to most student fabrication work.

How do I submit a 3D print or laser cut request?

Use the fabrication request form. Include your name, class, project, file, and any important instructions. Submit only when the file is ready for review.

Can I use the machines myself?

Only after completing the required safety expectations and receiving instructor approval. Some tools may always require direct supervision.

Why was my file rejected?

Common reasons include incorrect file type, wrong scale, unsafe geometry, missing dimensions, unsupported material, unclear instructions, or a design that does not match the project requirements.

What file types should I submit?

For 3D printing, submit the required 3D model or export file assigned by your instructor. For laser work, submit clean vector-ready geometry when requested. If you are unsure, ask before submitting.

How long will fabrication take?

Turnaround depends on file quality, material availability, machine queue, class priorities, and whether the file needs revision.

What should I do if my print or cut fails?

Document the failure, identify the likely cause, revise the design or settings, and resubmit only after making a clear improvement.

Can I bring my own material?

Ask first. Some materials are unsafe for certain machines, especially laser cutters. Unknown materials should never be used without approval.

How do I know when my item is ready?

Check the project status system or wait for instructor direction. Do not remove items from machines, shelves, or pickup areas unless they are yours and ready for pickup.
