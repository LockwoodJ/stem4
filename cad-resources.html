Before Submitting CAD

Run through these checks before requesting a 3D print, laser cut, or project review.

Model Check

  • All important dimensions are defined.
  • The model is oriented correctly for fabrication.
  • There are no floating, broken, or incomplete features.
  • File names clearly identify the course, project, and part.
  • The design matches the project constraints.

Drawing / Export Check

  • Drawings include correct views and dimensions.
  • Laser files use correct line types and clean geometry.
  • 3D print files are exported in the required format.
  • Assemblies show how parts fit together.
  • Special material or fabrication notes are included.
