Project Expectations

Strong projects show the full engineering process, not just the final product.

Every Project Should Include

  • Problem definition and constraints
  • Sketches, planning, or concept development
  • Engineering notebook evidence
  • Testing, feedback, or revision
  • Final reflection on what improved

Before Submitting

  • Check all file names and file types
  • Include screenshots, photos, or documentation
  • Make sure your work matches the project requirements
  • Submit fabrication files only when ready for review
