Aerospace Engineering Focus

Every unit connects engineering principles to aircraft, spacecraft, drones, launch systems, avionics, structures, and mission-ready design.

Project Overview

This project should show aerospace context, engineering process, evidence, testing, and improvement—not just a final product.

Challenge

Design Task

Create a working electronic/control system that reads an input, makes a decision, and produces an output related to an aerospace mission need.

Deliver

Required Evidence

Circuit diagram, code screenshot, sensor data, troubleshooting log, final demo, reflection.

Context

Aerospace Application

Connect the design to aircraft, spacecraft, drones, launch systems, mission equipment, ground support, or crew operations.

Required Engineering Evidence

  • Clear problem statement, target aerospace application, criteria, and constraints.
  • Sketches, CAD, diagrams, calculations, or circuit/code evidence as needed.
  • Prototype, model, system diagram, or test setup.
  • Testing data and observations from multiple trials when possible.
  • Design revisions based on evidence, not preference alone.
  • Final reflection explaining what improved and why.

Before Submission

  • Check the project requirements and rubric.
  • Confirm all files are named clearly and linked correctly.
  • Make sure the engineering notebook tells the full design story.
  • Use the fabrication request page only when files are ready for review.
