Unit Overview

Build and analyze systems made from multiple parts, focusing on fit, function, tolerances, constraints, and how components work together.

Unit Focus

Use these focus areas to guide your notebook entries, classwork, project planning, and reflection.

CAD Assemblies

Parts in a System

Model and assemble components so they align, fit, and function together.

Tolerances

Fit and Function

Understand how small dimensional differences affect movement, connection, and performance.

Testing

Assembly Performance

Use data and observations to evaluate how the system performs as a whole.

Skills and Deliverables

This unit builds the technical and documentation skills needed for successful project work.

Key Skills

  • Part modeling
  • Assembly constraints
  • Tolerances
  • Fit checks
  • Exploded views
  • Testing and revision

Evidence to Produce

  • CAD part models
  • Assembly evidence
  • Dimension and tolerance notes
  • Testing data
  • Design revisions
  • Final reflection

Suggested Workflow

Follow this sequence to move from learning to application and final reflection.

01
Analyze

Study how parts interact and what the assembly must do.

02
Model

Create or modify parts with accurate dimensions.

03
Assemble

Constrain parts and check fit relationships.

04
Test

Evaluate performance and identify problems.

05
Revise

Improve fit, stability, or function based on evidence.
