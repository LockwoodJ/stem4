Aerospace Engineering Focus

Every unit connects engineering principles to aircraft, spacecraft, drones, launch systems, avionics, structures, and mission-ready design.

Unit Focus

Sensors, Feedback, and Mission Control — Students use aerospace applications to make the engineering principles concrete, testable, and connected to real design decisions.

Understand

Aerospace Concepts

  • Circuits, sensors, and actuators
  • Feedback loops and control logic
  • Autonomous systems and troubleshooting
  • Data collection and calibration
  • Avionics and mission-control thinking
Apply

Technical Skills

  • Build a sensor-controlled circuit or robotic subsystem
  • Write or modify control logic
  • Calibrate sensor thresholds
  • Troubleshoot using data
  • Explain a feedback-controlled aerospace system
Document

Engineering Evidence

  • Circuit diagram or code screenshot
  • Sensor data table
  • Troubleshooting log
  • Control system test results
  • Final avionics demonstration reflection

Student Deliverables

  • Dated engineering notebook entries with sketches, data, screenshots, photos, calculations, and reflections.
  • Technical work showing aerospace context, design decisions, and evidence-based revision.
  • Prototype, model, diagram, test result, or analysis artifact connected to the unit focus.
  • Final reflection explaining what changed, why it changed, and how the evidence supports the decision.
