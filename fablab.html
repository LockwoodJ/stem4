Before You Use the FabLab

Good fabrication starts before the machine is turned on.

1. Get ApprovedComplete the required certification or safety expectations before using equipment.
2. Prepare FilesCheck dimensions, file type, scale, material choice, and machine readiness before submitting.
3. Track FeedbackWatch for revision requests, rejected files, or pickup instructions after your request is reviewed.

Common Fabrication Checks

  • Is the file named clearly?
  • Are the dimensions correct?
  • Is the design safe to fabricate?
  • Does the file match the project requirements?
  • Did you include any special instructions?

Ask Before...

  • Using unfamiliar tools or machines.
  • Changing machine settings.
  • Using new materials.
  • Running a file you did not create.
  • Removing another student's work from equipment.
